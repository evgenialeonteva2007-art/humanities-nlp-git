# 📖 Частотный анализатор текста (Humanities NLP)
Простой скрипт на Python для выделения ключевых слов в русскоязычных текстах. 
Разработан в рамках учебного курса.

## 📋 Что умеет
- Подсчёт частоты слов с исключением пунктуации
- Фильтрация стоп-слов (предлоги, союзы, местоимения)
- Вывод топ-N самых частотных лексических единиц
- Настройка параметров через аргументы функции

## ⚙️ Требования
- Python 3.8+
- Стандартные библиотеки: `re`, `collections` (установка не требуется)

## 🚀 Как запустить
python analyzer.py
